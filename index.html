The difference between paddings, margins and borders; padding is used to create space inside an element's content, inside of any defined borders, margins is used to create space around elements, outside of any defined borders. and borders is used to specify the style, width and color of an elements border. The total width or height can be calculated by the addition of border, width and padding with the exclusion of margin.
